## 🌟 Dream Project — FeedMe (Productive Social Media)
 
> This is the project I actually care about building.
 
Every platform today — YouTube, Instagram, Reels — decides what you watch based on what keeps you scrolling the longest. **FeedMe flips that.**
 
You pick your categories. The algorithm shows you only what you chose — ranked by your own watch behavior, not engagement bait. Coding, fitness, anime motivation, lifestyle — your feed, your rules.
 
**What's built so far:**
* Node.js + Express + **TypeScript** backend from scratch
* MongoDB with 5 collections — designed for optimized read/write performance
* JWT auth with access + refresh tokens
* Custom error handling — ApiError, asyncHandler, global error middleware
* Winston logger with log rotation
* Feed algorithm — cold start for new users, personalized scoring for returning users
**Tech:** MEAN stack · TypeScript end to end · YouTube + Instagram embeds
 
**Status:** Backend complete. Feed API + Angular frontend in progress.
 
> Open source — contributors welcome.
